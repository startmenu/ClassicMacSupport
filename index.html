欢迎造访本站!

如果你从犄角旮旯里翻出了一台老Macintosh电脑，先别急着扔掉。本站会为你提供有用的信息和资源，让你的老Mac电脑重获新生。

本站旨在分享老Mac使用经验和软件资源，以中文和Classic Mac OS资源为主。Classic Mac OS指的是Mac OS X 10.0之前的Mac系统，从System Software 1.0到9.2.2，但本站侧重于提供System 7到Mac OS 9的信息和资源。当然，本站也提供早期Mac OS X（含早期Intel版）的资源（建设中的“Mac OS X专区”）。

为了在老机器和老版本浏览器上获得较流畅的浏览体验，本站网页一切从简，没有复杂图片，没有https，没有script，采用GB2312编码。
建议使用Internet Explorer 4.0或Netscape 4.0（均支持68k Mac）及更高版本的浏览器访问本站。
屏幕分辨率建议800x600或1024x768。分辨率太高（或许是超过1280x800）可能会带来些不愉快的体验，当然也可以缩小浏览器窗口来弥补。
