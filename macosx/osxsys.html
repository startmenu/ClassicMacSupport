如何选择？

如果是老Intel Mac用户，建议直接选择Mac OS X 10.6系统，并更新到10.6.8。设备支持的情况下，可以选择更新版的Mac OS X，如10.7。Mac OS X 10.4的Intel版本并不推荐使用，因为是随机器附带的安装盘，并不通用，且软件资源也不算丰富。

PowerPC用户则认准三个版本――10.2，10.4和10.5，根据硬件配置来选择。Mac OS X 10.4.11和10.5.8都有较为丰富的软件支持。10.4适合大部分PowerPC Mac，10.5适合高配置的G4以及G5机型。

Mac OS X 10.0 Cheetah

概述

Mac OS X 10.0发布于2001年3月份，最新版本是10.0.4（2001年6月）。Darwin版本 1.3.1。不推荐使用此版本，建议至少使用10.2版。

Mac OS X 10.4 Tiger

概述

Mac OS X 10.4发布于2005年4月，是最经典的Mac OS X版本之一。最新版本10.4.11，Darwin版本8.0到8.11。10.4.4版开始支持Intel处理器，但Intel版只有随机器附带的版本，零售版只有PowerPC的。

Mac OS X 10.4是最后带有Classic环境的版本，可以在PowerPC平台上运行Classic Mac OS应用程序（Mac OS 9及更早版本的应用程序）。但10.4的Classic环境的性能并不如10.3版本的。Intel版本的10.4则不支持Classic环境，无法运行Classic应用程序。


资源页面：

The Tiger Thread（资源帖）

兼容性

Mac OS X 10.4安装介质只有DVD，而且要求具有内置FireWire（火线）的Mac机型。至少需要2000年的PowerBook G3机型（Pismo），iMac DV 机型，iBook （FireWire）机型，Power Macintosh 蓝白机。Intel机型最高支持到：

  • iMac （Mid 2007）
  • MacBook (Mid 2007)
  • MacBook Pro (Late 2007)
  • Mac mini (Mid 2007)
  • Mac Pro (8-core, 2007)
实际也可以在早期2008年的机型上使用，但可能没有部分驱动。

Mac OS X 10.5 Leopard

概述

Mac OS X 10.5发布于2007年10月，拥有全新的外观。不少底层技术也有较大改进。最新版本10.5.8，Darwin版本9.0到9.8。Mac OS X 10.5同时支持PowerPC和Intel Mac机型，同时也是最后支持PowerPC的Mac OS X版本。Mac OS X 10.5在PowerPC平台的性能表现不如之前的10.4版本，在Intel平台上则比10.4版更优。

兼容性

Mac OS X 10.5要求至少PowerPC G4 867MHz的Mac机型，不兼容PowerPC G3。低于867MHz的G4机型可以借助Leopard Assistant来安装Mac OS X 10.5。要获得较好的使用体验，建议使用Intel机型安装，PowerPC机型建议至少是G4 1.2GHz并且显卡至少是ATI 9500或NVIDIA 6000系列。

Mac OS X 10.6 Snow Leopard

概述

Mac OS X 10.6发布于2009年8月，是10.5的优化改进版本，界面相当于10.5有少许变化。最新版本10.6.8，Darwin版本从10.0到10.8。Mac OS X 10.6系统从10.6.6开始支持Mac App Store。Mac OS X 10.6不兼容PowerPC Mac，却是最后一个能运行PowerPC应用程序的版本。安装时建议勾选Rosetta和QuickTime 7。Mac OS X 10.6移除了AppleTalk协议支持，而且共享的文件和文件夹不能在Mac OS 9中访问。

Mac OS X 10.7 Lion

概述

Mac OS X 10.7发布于2011年7月，是首个融入iOS设计的Mac OS X版本，首次推出Launchpad和iCloud。最新版本10.7.5，Darwin版本从11.0到11.5。Mac OS X 10.7移除了Rosetta，不再支持运行PowerPC应用程序。

OS X 10.8 Mountain Lion

概述

OS X 10.8发布于2012年7月，是最后一个以大学猫科动物命名的系统版本，同时系统更名为OS X，直到10.12才更名为macOS。最新版本10.8.5，Darwin版本从12.0到12.5。

OS X 10.8继续放弃了一些旧机型的支持，比如所有Intel GMA系列显卡的机型。
