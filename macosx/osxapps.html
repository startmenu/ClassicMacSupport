实用工具

The Unchiver 老版本（Universal）兼容Mac OS X 10.4.11到10.6.8，是一款解压缩软件，支持多种格式的压缩文件。


Roxio Toast是Mac平台老牌的光盘刻录和光盘镜像工具。老版本可以在Macintosh Garden上找到。


Burn 是一款Mac OS X平台的刻录软件，能支持刻录bin/cue格式的镜像。其2.5.1版兼容PowerPC Mac。2.6版开始可能要求OS X 10.9。


Fun Input Toy 输入法是专为Mac OS X开发的中文输入法。1.5.1版最后支持Mac OS X 10.4.11（Intel和PPC），1.9.1版最后支持Mac OS X 10.5.8 PPC，2.3版本支持Intel Mac OS X 10.5.8-10.7。可在网盘中找到。


多媒体

VLC 是较为全能的播放器，支持大多格式的视频音频文件。它的网站上也给出了不同老版本Mac OS X能支持的VLC版本及下载。


CorePlayer 是PowerPC平台较好的H.264视频播放器，能在PowerPC G4 1.25GHz上较流畅播放部分720p视频。也支持Intel Mac。


Perian 能够让QuickTime支持更多格式的媒体文件。Perian 1.2.3 (Universal)支持Mac OS X 10.4.7及更高版本，但不兼容OS X 10.8及更高版本。Perian现已停止维护。
